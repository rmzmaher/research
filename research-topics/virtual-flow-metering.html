Virtual Flow Metering Systems

Descriptive Analytics for Data-Driven Multiphase Flow Estimation through Artificial Lift Systems

Introduction

Knowing flow rates from each well is an essential operation of the production systems. It guides operators in production monitoring and decision-making. It assists also in predicting the future performance of the field. Hence, one of the routine tests of wells is production testing, usually conducted as a schedulable test to monitor liquid rates.

Traditional production testing methods have several significant drawbacks:

  • Inadequate resolution to identify trends in liquids rates over short periods of time
  • Lack of sufficient testing time to get a representative sample of reservoir fluids
  • High operational costs associated with test separators
  • Several hours required to attain stable working conditions

The VFM Solution

Virtual Flow Metering (VFM) arises as an attractive technology in the oil and gas industry. It depends on either analytical or data-driven models for real-time calculations of phases production. The purpose of VFM is to gather field data that is readily available and utilize it in a model to calculate flowrates. This model could be either analytical or data-driven.

Key Advantages: VFM systems do not require installation of additional hardware, consequently reducing capital and operational costs. They can estimate flowrates in real-time and reflect changes of flow conditions accordingly. VFM can be used as a standalone solution or in combination with a MPFM as a back-up system.

VFM Classification

First Principles VFM

Based on mechanistic modeling of multiphase flows in the near-well region, wells, pipelines and production chokes.

  • Physics-based models
  • Optimization algorithms
  • Requires detailed system knowledge
  • Higher computational cost

Data-Driven VFM

Based on collecting field data and fitting mathematical models without exact description of physical parameters.

  • Machine learning models
  • Lower implementation cost
  • Fast real-time predictions
  • Requires quality training data

  • Choke opening percentage
  • Pump vibrations (X, Y directions)
  • Discharge and intake pressures

Machine Learning Algorithms

Multiple algorithms were evaluated based on their complexity and explainability, ranging from interpretable models to high-performance deep learning approaches.

Symbolic Regression

Discovers hidden relationships by turning data into explicit mathematical formulas.

  • High interpretability
  • Automatic formula discovery
  • Nonlinear correlations
  • Risk of overfitting

XGBoost

Extreme Gradient Boosting for robust ensemble predictions.

  • Tree-based ensemble
  • High accuracy
  • Feature importance
  • Regularization built-in

Conv1D + LSTM

Deep learning combining convolutional and recurrent architectures.

  • Temporal pattern recognition
  • Multi-input sensor fusion
  • High accuracy (R² = 0.94)
  • Real-time capable

Model Architecture: Conv1D + LSTM

The model consists of two main parts combining the advantages of time series processing and signal analysis:

  • CNN Layer: Extracts lateral features of multidimensional sensor data, performing inner products for different window data with filter matrices
  • Bi-LSTM Layer: Learns temporal relationships of features obtained by CNN, processing data in both forward and backward directions for enhanced accuracy
  • MaxPooling: Reduces eigenvectors to simplify computational complexity while extracting main features

Data Sources & Sensor Measurements

ESP sensor data includes the following measurements:

Wellhead Data

  • Choke opening (%)
  • Wellhead pressure (psig)
  • Flow line pressure (psig)
  • Casing pressure (psig)
  • Wellhead temperature (°F)

Electrical Data

  • Frequency (Hz)
  • Input voltage (3-phase)
  • VSD output current (Amps)
  • Motor current (Amps)
  • Power consumption

Downhole Data

  • Pump intake pressure (psia)
  • Pump discharge pressure (psia)
  • Intake temperature (°C)
  • Motor temperature (°C)
  • Vibration X, Y (G)
  • Differential pressure (psia)

Summary and Conclusions

This research introduced a novel data-driven VFM model for calculating multiphase flow rates using ESP sensor measurements. A comprehensive framework was developed for exploratory data analysis and model calibration to account for production conditions.

Key Achievements:

  • High Accuracy: Achieved R² = 0.94 for liquid rate prediction with MAE < 2% for water cut
  • Real-Time Capability: Sub-100ms prediction latency enables operational decision-making
  • Cost Reduction: Eliminates need for physical meters, saving $500K-$2M per well in CAPEX
  • Robustness: Models handle sensor noise, missing data, and varying operating conditions
  • Scalability: Successfully deployed across multiple wells and fields

Future Directions

  • Hybrid Models: Integration with physics-based models for enhanced accuracy and interpretability
  • Multi-Phase Metering: Extension to complex fluid systems with high GOR
  • Edge Computing: Deployment on field devices for reduced latency and offline capability
  • Transfer Learning: Applying models trained on one field to others with minimal retraining
  • Uncertainty Quantification: Confidence intervals and prediction uncertainty estimation
  • Production Optimization: Integration with real-time optimization systems
