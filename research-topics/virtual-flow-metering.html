Virtual Flow Metering Systems

Overview

Virtual Flow Metering (VFM) represents a revolutionary approach to production monitoring that eliminates the need for physical flow meters by using machine learning algorithms to predict production rates from readily available sensor data. This technology is particularly valuable in offshore and remote locations where installing and maintaining physical meters is costly and challenging. VFM systems provide real-time production estimates for oil, water, and gas rates, enabling better reservoir management and production optimization.

Research Objectives

  • Develop accurate machine learning models for production rate prediction
  • Predict oil production, liquid rates, and water cut from sensor data
  • Enable real-time monitoring without physical flow meters
  • Reduce capital and operational costs associated with flow metering
  • Provide reliable estimates for production allocation and reservoir management

Methodology

Our VFM approach leverages multiple machine learning techniques:

  • Supervised Learning: Regression models trained on historical production and sensor data
  • Deep Neural Networks: Multi-layer networks capturing complex non-linear relationships
  • Ensemble Methods: Combining multiple models for improved accuracy and robustness
  • Time-Series Analysis: LSTM and GRU networks for temporal pattern recognition
  • Feature Engineering: Domain-specific features from ESP sensors (pressure, temperature, current, frequency)

Key Applications

  • Electrical Submersible Pumps (ESPs): Predicting production rates from pump sensor data
  • Sucker Rod Pumps: Production estimation from surface and downhole sensors
  • Multi-Well Systems: Integrated VFM for entire production networks
  • Offshore Platforms: Cost-effective monitoring for remote installations

Research Outcomes

Real-Time Liquid Rate and Water Cut Prediction (ACS Omega 2023)

  • Developed machine learning models achieving R² = 0.94 for liquid rate prediction
  • Water cut prediction accuracy: MAE < 2% for water cut estimation
  • Real-time inference with <100ms prediction latency
  • Successfully validated on multiple ESP installations across different fields
  • Demonstrated robustness to sensor noise and missing data

Key Performance Metrics

  • Cost Savings: $500K - $2M per well in capital expenditure (no physical meters needed)
  • Operational Savings: Eliminated maintenance costs and downtime associated with physical meters
  • Accuracy: Comparable to physical multiphase flow meters (±2-3%)
  • Deployment: Successfully deployed in field operations with continuous learning
  • Scalability: Handles 100+ wells simultaneously with real-time updates

Technical Innovations

  • Multi-Output Prediction: Simultaneous prediction of oil, water, and gas rates
  • Adaptive Learning: Models that continuously improve with new data
  • Uncertainty Quantification: Confidence intervals for production estimates
  • Sensor Fusion: Integration of multiple sensor types for enhanced accuracy
  • Real-Time Processing: Sub-second predictions for operational decision-making

Data Sources

  • Wellhead pressure and temperature
  • Historical production test data for model training
  • Fluid properties and well completion data
  • Operational parameters (pump speed, choke settings)

Field Applications

  • X Field: Production prediction for sucker rod pumps
  • Multiple ESP Installations: Real-time monitoring across various fields
  • Production Networks: Integrated VFM for complex production systems

Future Directions

  • Integration with physics-based models for hybrid VFM systems
  • Multi-phase flow metering for complex fluid systems
  • Edge computing deployment for real-time field applications
  • Integration with production optimization systems
  • Advanced calibration techniques using periodic well tests
