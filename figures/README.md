## How to Extract Figures from Your Papers

1. **From Published Papers:**
   - Download PDFs from publisher websites using DOIs
   - Extract figures using PDF tools or screenshot
   - Save as PNG for static images, GIF for animations

2. **From Open Access Papers:**
   - Many papers have supplementary materials with high-resolution figures
   - Check publisher websites for "Supporting Information" or "Supplementary Materials"

3. **Creating Animations:**
   - Use tools like Python (matplotlib, plotly) to recreate animations
   - Export as GIF or MP4 format
